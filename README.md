# PrizeRink IQ

PrizeRink IQ is a curling strategy companion for players and fans. It includes an interactive house board, shot simulator, match scorekeeper, match journal, accuracy tracking, lineup notes, beginner strategy guides, and local-first privacy settings.

## Build

- Xcode project: `PrizeRinkIQ.xcodeproj`
- Scheme: `PrizeRinkIQ`
- Bundle ID: `com.prizerink.iq`
- App Store Apple ID: `6773018747`
- Minimum iOS: 17.0

Codemagic is configured in `codemagic.yaml` to validate Debug and Release builds, sign with App Store profiles, build an IPA, and upload to TestFlight.
